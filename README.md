# pdfsplit

A Clojure library that splits a pdf into single pages

## Usage

[![Clojars Project](https://img.shields.io/clojars/v/pdfsplit.svg)](https://clojars.org/pdfsplit)

    (:require [pdfsplit.core :refer [split]])

    (split "path/to/file.pdf")
